# RetroTUI — Roadmap

**Objetivo:** Un entorno de escritorio estilo Windows 3.1 completamente funcional para la terminal. Sin X11. Sin Wayland. Solo curses, una TTY y vibes.

**Meta de release:** llegar a **v0.9.8 feature complete**, usar **v0.9.9 exclusivamente para bugtest/estabilización**, y lanzar **v1.0.0 sin bugs críticos conocidos**.

**Estado actual:** v0.9.4 hardening cerrado. v0.9.5 (Terminal 2D buffer) en progreso: `TerminalScreenBuffer` y `TerminalScreen` ya viven
